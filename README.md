# HEMS-Energy-Data-Science
A data-driven Home Energy Management System (HEMS) to optimize electricity costs. Features an XGBoost demand forecasting model (15.5% NRMSE) and a smart battery control algorithm that maximizes solar self-consumption and generates profit.
